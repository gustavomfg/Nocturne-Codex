# v0.8.0-beta — 2026-07-16

Compatibilidade Codex CLI: mínima 0.0.0.

## Fundacao Provider Agnostic

- Provider Registry inicial comeca vazio, registra somente configuracoes explicitas com adapters substituiveis.
- Model Registry normalizado identifica modelos por `providerId + modelId`.
- Task Builder cria solicitacoes provider-independent com contexto classificado.
- AI Orchestrator executa tarefas normalizadas por adapters substituiveis.
- Adapter OpenAI-compatible reutilizavel valida endpoints remotos e loopback.
- Cofre de creditos do main process persiste ciphertext atomico com permissoes 0600.
- Schema SQLite 9 e 10 adicionam configuracoes de Provider e bindings de modelos.
